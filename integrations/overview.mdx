## Integration Features

Each integration provides:

### Autologging
Automatically capture metrics, parameters, and models without explicit logging code:

```python
import mlflow

# Enable autologging for your framework
mlflow.pytorch.autolog()
mlflow.sklearn.autolog()
mlflow.tensorflow.autolog()
```

### Model Logging
Log models in MLflow's standard format for easy deployment:

```python
import mlflow.pytorch

with mlflow.start_run():
    # Train your model
    model = train_model()
    
    # Log the model
    mlflow.pytorch.log_model(model, "model")
```

### Tracing
Capture execution traces for LLM applications:

```python
import mlflow

mlflow.langchain.autolog(log_traces=True)
mlflow.openai.autolog(log_traces=True)
```

### Deployment
Deploy models as REST APIs or batch inference jobs:

```bash
mlflow models serve -m runs:/<run_id>/model -p 5000
```

## Getting Started

To get started with a specific integration, select the framework from the list above or install the required dependencies:

```bash
# Install MLflow with specific integrations
pip install mlflow[extras]

# Or install framework separately
pip install mlflow torch pytorch-lightning
pip install mlflow scikit-learn
pip install mlflow tensorflow
pip install mlflow transformers
pip install mlflow langchain openai
```
