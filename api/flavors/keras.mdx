title: mlflow.keras

The `mlflow.keras` module provides an API for logging and loading Keras models (Keras 3+). For Keras 2.x (bundled with TensorFlow), use `mlflow.tensorflow` instead.

<Note>
This flavor supports Keras 3+. For Keras versions < 3, the module redirects to `mlflow.tensorflow`.
</Note>

## Saving Models

### save_model()

Save a Keras model along with metadata to the local file system.

```python
mlflow.keras.save_model(
    model,
    path,
    save_exported_model=False,
    conda_env=None,
    mlflow_model=None,
    signature=None,
    input_example=None,
    pip_requirements=None,
    extra_pip_requirements=None,
    save_model_kwargs=None,
    metadata=None
)
```

<ParamField path="model" type="keras.Model" required>
  An instance of `keras.Model` to be saved.
</ParamField>

<ParamField path="path" type="str" required>
  Local path where the MLflow model is to be saved.
</ParamField>

<ParamField path="save_exported_model" type="bool" default="False">
  If True, save Keras model in exported model format, otherwise save in `.keras` format. See [Keras serialization guide](https://keras.io/guides/serialization_and_saving/).
</ParamField>

<ParamField path="signature" type="ModelSignature">
  Model signature describing model inputs and outputs. All fields must be TensorSpec with first dimension as -1.
</ParamField>

<ParamField path="input_example" type="np.ndarray | pd.DataFrame">
  Example input for model testing and signature inference.
</ParamField>

<ParamField path="save_model_kwargs" type="dict">
  Dict of kwargs to pass to `keras.Model.save` method.
</ParamField>

### log_model()

Log a Keras model along with metadata to MLflow.

```python
mlflow.keras.log_model(
    model,
    artifact_path=None,
    save_exported_model=False,
    conda_env=None,
    signature=None,
    input_example=None,
    registered_model_name=None,
    await_registration_for=300,
    pip_requirements=None,
    extra_pip_requirements=None,
    save_model_kwargs=None,
    metadata=None,
    name=None,
    params=None,
    tags=None,
    model_type=None,
    step=0,
    model_id=None
)
```

<ParamField path="model" type="keras.Model" required>
  The Keras model to be logged.
</ParamField>

<ParamField path="name" type="str">
  Name of the logged model.
</ParamField>

<ParamField path="save_exported_model" type="bool" default="False">
  If True, save in exported model format, otherwise save in `.keras` format.
</ParamField>

<ParamField path="registered_model_name" type="str">
  If set, create a model version under this registered model name.
</ParamField>

**Returns:** `ModelInfo` instance containing metadata of the logged model.

## Loading Models

### load_model()

Load a Keras model from a local file or a run.

```python
mlflow.keras.load_model(
    model_uri,
    dst_path=None
)
```

<ParamField path="model_uri" type="str" required>
  The location, in URI format, of the MLflow model. Supported URI schemes:
  - `/Users/me/path/to/local/model`
  - `s3://my_bucket/path/to/model`
  - `runs:/<mlflow_run_id>/run-relative/path/to/model`
  - `models:/<model_name>/<model_version>`
</ParamField>

**Returns:** A Keras model instance.

## Autologging

### autolog()

Enables autologging for Keras.

```python
mlflow.keras.autolog(
    log_models=True,
    log_datasets=True,
    disable=False,
    exclusive=False,
    disable_for_unsupported_versions=False,
    silent=False,
    registered_model_name=None,
    log_input_examples=False,
    log_model_signatures=True,
    extra_tags=None
)
```

<ParamField path="log_models" type="bool" default="True">
  If True, trained models are logged as MLflow model artifacts.
</ParamField>

<ParamField path="log_input_examples" type="bool" default="False">
  If True, input examples from training datasets are collected and logged.
</ParamField>

<ParamField path="log_model_signatures" type="bool" default="True">
  If True, ModelSignatures are collected and logged with model artifacts.
</ParamField>

## Helper Functions

### get_default_pip_requirements()

Returns default pip requirements for Keras models.

```python
mlflow.keras.get_default_pip_requirements()
```

**Returns:** List of default pip requirements.

### get_default_conda_env()

Returns default Conda environment for Keras models.

```python
mlflow.keras.get_default_conda_env()
```

**Returns:** Dictionary representing the default Conda environment.

## Example

```python
import keras
import mlflow
import mlflow.keras

# Define a simple model
model = keras.Sequential([
    keras.Input([28, 28, 3]),
    keras.layers.Flatten(),
    keras.layers.Dense(2)
])

# Log the model
with mlflow.start_run():
    mlflow.keras.log_model(model, name="keras_model")

# Load and use the model
loaded_model = mlflow.keras.load_model(f"runs:/{run.info.run_id}/keras_model")
```
