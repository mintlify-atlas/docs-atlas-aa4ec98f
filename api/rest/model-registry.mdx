## Create Registered Model

Create a new registered model with a unique name.

<CodeGroup>
```bash cURL
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-model",
    "description": "My production model",
    "tags": [
      {"key": "team", "value": "ml-platform"},
      {"key": "stage", "value": "production"}
    ]
  }' \
  http://localhost:5000/api/2.0/mlflow/registered-models/create
```

```python Python
import requests

response = requests.post(
    "http://localhost:5000/api/2.0/mlflow/registered-models/create",
    json={
        "name": "my-model",
        "description": "My production model",
        "tags": [
            {"key": "team", "value": "ml-platform"},
            {"key": "stage", "value": "production"}
        ]
    }
)
print(response.json())
```
</CodeGroup>

**Endpoint:** `POST /api/2.0/mlflow/registered-models/create`

### Request Body

<ParamField body="name" type="string" required>
  Unique name for the registered model.
</ParamField>

<ParamField body="description" type="string">
  Optional description for the registered model.
</ParamField>

<ParamField body="tags" type="array">
  Additional metadata for the registered model.
  
  <Expandable title="Tag object">
    <ParamField body="key" type="string" required>
      Tag key.
    </ParamField>
    <ParamField body="value" type="string" required>
      Tag value.
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="registered_model" type="object">
  <Expandable title="RegisteredModel object">
    <ResponseField name="name" type="string">
      Unique name for the model.
    </ResponseField>
    <ResponseField name="creation_timestamp" type="integer">
      Unix timestamp in milliseconds when created.
    </ResponseField>
    <ResponseField name="last_updated_timestamp" type="integer">
      Unix timestamp in milliseconds of last update.
    </ResponseField>
    <ResponseField name="description" type="string">
      Model description.
    </ResponseField>
    <ResponseField name="latest_versions" type="array">
      Collection of latest model versions for each stage.
    </ResponseField>
    <ResponseField name="tags" type="array">
      Array of tag objects.
    </ResponseField>
  </Expandable>
</ResponseField>

```json Response Example
{
  "registered_model": {
    "name": "my-model",
    "creation_timestamp": 1678886400000,
    "last_updated_timestamp": 1678886400000,
    "description": "My production model",
    "latest_versions": [],
    "tags": [
      {"key": "team", "value": "ml-platform"}
    ]
  }
}
```

## Search Registered Models

Search for registered models matching filter criteria.

<CodeGroup>
```bash cURL
curl "http://localhost:5000/api/2.0/mlflow/registered-models/search?filter=name%20LIKE%20%27my-%25%27&max_results=100"
```

```python Python
import requests

response = requests.get(
    "http://localhost:5000/api/2.0/mlflow/registered-models/search",
    params={
        "filter": "name LIKE 'my-%'",
        "max_results": 100
    }
)
print(response.json())
```
</CodeGroup>

**Endpoint:** `GET /api/2.0/mlflow/registered-models/search`

### Query Parameters

<ParamField query="filter" type="string">
  Filter expression. Example: `name LIKE 'my-model%'`
  
  Supported operators: `=`, `!=`, `LIKE`, `ILIKE`
</ParamField>

<ParamField query="max_results" type="integer">
  Maximum number of models to return. Default is 100, maximum is 1000.
</ParamField>

<ParamField query="order_by" type="array">
  List of columns for ordering. Can include `name` and `last_updated_timestamp` with optional `DESC` or `ASC`.
</ParamField>

<ParamField query="page_token" type="string">
  Token for pagination.
</ParamField>

### Response

<ResponseField name="registered_models" type="array">
  Array of registered model objects.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token for retrieving the next page.
</ResponseField>

## Create Model Version

Create a new model version from a run's artifacts.

<CodeGroup>
```bash cURL
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-model",
    "source": "runs:/abc123def456/model",
    "run_id": "abc123def456",
    "description": "Model from experiment run",
    "tags": [
      {"key": "validation_status", "value": "approved"}
    ]
  }' \
  http://localhost:5000/api/2.0/mlflow/model-versions/create
```

```python Python
import requests

response = requests.post(
    "http://localhost:5000/api/2.0/mlflow/model-versions/create",
    json={
        "name": "my-model",
        "source": "runs:/abc123def456/model",
        "run_id": "abc123def456",
        "description": "Model from experiment run",
        "tags": [
            {"key": "validation_status", "value": "approved"}
        ]
    }
)
print(response.json())
```
</CodeGroup>

**Endpoint:** `POST /api/2.0/mlflow/model-versions/create`

### Request Body

<ParamField body="name" type="string" required>
  Name of the registered model.
</ParamField>

<ParamField body="source" type="string" required>
  URI indicating the location of model artifacts. Typically `runs:/<run_id>/model`.
</ParamField>

<ParamField body="run_id" type="string">
  MLflow run ID that generated the model.
</ParamField>

<ParamField body="description" type="string">
  Optional description for the model version.
</ParamField>

<ParamField body="tags" type="array">
  Additional metadata for the model version.
</ParamField>

<ParamField body="run_link" type="string">
  Direct link to the run that generated this version.
</ParamField>

### Response

<ResponseField name="model_version" type="object">
  <Expandable title="ModelVersion object">
    <ResponseField name="name" type="string">
      Model name.
    </ResponseField>
    <ResponseField name="version" type="string">
      Model version number.
    </ResponseField>
    <ResponseField name="creation_timestamp" type="integer">
      Unix timestamp when created.
    </ResponseField>
    <ResponseField name="current_stage" type="string">
      Current deployment stage (None, Staging, Production, Archived).
    </ResponseField>
    <ResponseField name="source" type="string">
      URI of source model artifacts.
    </ResponseField>
    <ResponseField name="run_id" type="string">
      Associated run ID.
    </ResponseField>
    <ResponseField name="status" type="string">
      Version status: PENDING_REGISTRATION, FAILED_REGISTRATION, or READY.
    </ResponseField>
  </Expandable>
</ResponseField>

## Transition Model Version Stage

```bash cURL
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-model",
    "version": "1",
    "stage": "Production",
    "archive_existing_versions": true
  }' \
  http://localhost:5000/api/2.0/mlflow/model-versions/transition-stage
```

```python Python
import requests

response = requests.post(
    "http://localhost:5000/api/2.0/mlflow/model-versions/transition-stage",
    json={
        "name": "my-model",
        "version": "1",
        "stage": "Production",
        "archive_existing_versions": True
    }
)
```
</CodeGroup>

**Endpoint:** `POST /api/2.0/mlflow/model-versions/transition-stage`

### Request Body

<ParamField body="name" type="string" required>
  Model name.
</ParamField>

<ParamField body="version" type="string" required>
  Model version number.
</ParamField>

<ParamField body="stage" type="string" required>
  Target stage: `None`, `Staging`, `Production`, or `Archived`.
</ParamField>

<ParamField body="archive_existing_versions" type="boolean" required>
  If true, archive all existing versions in the target stage. Required parameter.
