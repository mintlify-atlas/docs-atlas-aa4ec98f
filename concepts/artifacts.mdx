## What are Artifacts?

Artifacts are files or directories associated with an MLflow run. They can include:

- Trained models (serialized model files)
- Data files (datasets, feature sets)
- Images and visualizations (plots, charts, diagrams)
- Metadata files (configuration, metrics summaries)
- Text files (logs, reports, notes)
- Any other file output from your ML code

Artifacts are stored in a backend-agnostic way, supporting local filesystems, cloud storage (S3, Azure Blob, GCS), and other storage systems.

## Artifact Storage

### Artifact URI

Each run has an `artifact_uri` that points to the root location where artifacts are stored:

```python
import mlflow

with mlflow.start_run() as run:
    # Get the artifact URI for this run
    artifact_uri = run.info.artifact_uri
    print(f"Artifacts will be stored at: {artifact_uri}")
    
    # Can also use the fluent API
    artifact_uri = mlflow.get_artifact_uri()
    
    # Get URI for specific artifact
    model_uri = mlflow.get_artifact_uri("model.pkl")
```

Common artifact URI formats:
- Local: `file:///path/to/mlruns/0/abc123/artifacts`
- S3: `s3://bucket-name/path/to/artifacts`
- Azure: `wasbs://container@account.blob.core.windows.net/path`
- GCS: `gs://bucket-name/path/to/artifacts`
- HDFS: `hdfs://namenode:port/path/to/artifacts`

### Configuring Artifact Storage

Set the artifact location when creating an experiment:

```python
import mlflow

# Create experiment with custom artifact location
experiment_id = mlflow.create_experiment(
    name="my-experiment",
    artifact_location="s3://my-bucket/mlflow-artifacts"
)

mlflow.set_experiment(experiment_id=experiment_id)
```

## Logging Artifacts

### Log Single Files

```python
import mlflow

with mlflow.start_run():
    # Log a single file
    mlflow.log_artifact("model.pkl")
    
    # Log to a specific subdirectory
    mlflow.log_artifact("confusion_matrix.png", "plots")
    # Stored at: artifacts/plots/confusion_matrix.png
```

### Log Directories

```python
import mlflow

with mlflow.start_run():
    # Log entire directory
    mlflow.log_artifacts("output_files/")
    
    # Log directory to specific path
    mlflow.log_artifacts("checkpoints/", "model_checkpoints")
    # Contents stored at: artifacts/model_checkpoints/
```

```python
import mlflow
```

### Log Figures

```python

import matplotlib.pyplot as plt

with mlflow.start_run():
    # Create matplotlib figure
```

## Artifact Repositories

MLflow uses artifact repositories to abstract storage backends:

```python
from mlflow.tracking.artifact_utils import get_artifact_repository

# Get artifact repository for a run
artifact_repo = get_artifact_repository(
    artifact_uri="runs:/abc123"
)

# Download artifacts
local_path = artifact_repo.download_artifacts("model")

# List artifacts
artifacts = artifact_repo.list_artifacts("plots")

# Log artifacts (if you have write access)
artifact_repo.log_artifact("local_file.txt", "remote_path")
```

## Artifact URIs

MLflow supports various URI schemes for referencing artifacts:

### Runs URI

Reference artifacts from a specific run:

```python
# Format: runs:/<run_id>/<artifact_path>
artifact_uri = "runs:/abc123def456/model.pkl"
artifact_uri = "runs:/abc123def456/plots/confusion_matrix.png"
```

### Models URI

Reference artifacts from registered models:

```python
# By version
model_uri = "models:/my-model/1"

# By stage
model_uri = "models:/my-model/Production"

# By alias
model_uri = "models:/my-model@champion"

# With specific artifact path
artifact_uri = "models:/my-model/1/path/to/artifact"
```

### Cloud Storage URIs

Direct cloud storage references:

```python
# S3
s3_uri = "s3://my-bucket/path/to/artifact"

# Azure Blob Storage
azure_uri = "wasbs://container@account.blob.core.windows.net/path"

# Google Cloud Storage
gcs_uri = "gs://my-bucket/path/to/artifact"
```

## Async Artifact Logging

For better performance, enable asynchronous artifact logging:

```python
import mlflow
import os

# Enable async artifact logging
os.environ["MLFLOW_ENABLE_ASYNC_LOGGING"] = "true"

with mlflow.start_run():
    # Artifacts are logged asynchronously in background
    for i in range(100):
        mlflow.log_artifact(f"output_{i}.txt")
    
    # Flush to ensure all artifacts are uploaded
    mlflow.flush_artifact_async_logging()
```

## Working with Large Artifacts

### Chunked Downloads

For large files, artifacts can be downloaded in chunks:

```python
from mlflow.artifacts import download_artifacts

# Download large model
local_path = download_artifacts(
    artifact_uri="runs:/abc123/large_model",
    dst_path="./models/"
)
```

### Streaming Logs

Stream logs in real-time during training:

```python
import mlflow
import time

with mlflow.start_run():
    # Stream logs to artifact
    with mlflow.log_stream("training.log") as stream:
        for epoch in range(100):
            log_message = f"Epoch {epoch}: loss=0.{100-epoch}\n"
            stream.write(log_message)
            time.sleep(0.1)
```

## Best Practices

<AccordionGroup>
  <Accordion title="Organize Artifacts in Subdirectories">
    Use meaningful subdirectories to organize different types of artifacts:
    
    ```python
    mlflow.log_artifact("model.pkl", "models")
    mlflow.log_artifact("plot.png", "visualizations")
    mlflow.log_artifact("metrics.json", "results")
    mlflow.log_artifact("config.yaml", "configs")
    ```
  </Accordion>
  
  <Accordion title="Use Appropriate Logging Functions">
    Choose the right function for your artifact type:
    
    ```python
    # For dictionaries/configs
    mlflow.log_dict(config, "config.json")
    
    # For text
    mlflow.log_text(notes, "notes.txt")
    
    # For plots
    mlflow.log_figure(fig, "plot.png")
    
    # For tabular data
    mlflow.log_table(df, "results.json")
    ```
  </Accordion>
  
  <Accordion title="Clean Up Local Files">
    MLflow logs artifacts by copying them. Clean up temporary files after logging:
    
    ```python
    import os
    import mlflow
    from mlflow.utils.file_utils import TempDir
    
    with mlflow.start_run():
        with TempDir() as tmp:
            output_file = tmp.path("output.txt")
            # Generate artifact
            with open(output_file, "w") as f:
                f.write("results")
            # Log artifact
            mlflow.log_artifact(output_file)
        # Temp directory automatically cleaned up
    ```
  </Accordion>
  
  <Accordion title="Enable Async Logging for Performance">
    When logging many artifacts, use async logging:
    
    ```python
    import os
    os.environ["MLFLOW_ENABLE_ASYNC_LOGGING"] = "true"
    
    # Your logging code...
    
    # Flush before ending run
    mlflow.flush_artifact_async_logging()
    ```
  </Accordion>
</AccordionGroup>

## Common Patterns

<CodeGroup>
```python Save Model Artifacts
import mlflow
import joblib

with mlflow.start_run():
    # Train model
    model = train_model()
    
    # Save and log model
    joblib.dump(model, "model.pkl")
    mlflow.log_artifact("model.pkl", "model")
    
    # Log model metadata
    mlflow.log_dict({
        "model_type": "RandomForest",
        "n_estimators": 100
    }, "model/metadata.json")
```

```python Save Training Plots
import mlflow
import matplotlib.pyplot as plt

with mlflow.start_run():
    history = train_model()
    
    # Create plots
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))
    
    ax1.plot(history["loss"])
    ax1.set_title("Training Loss")
    
    ax2.plot(history["accuracy"])
    ax2.set_title("Training Accuracy")
    
    # Log plots
    mlflow.log_figure(fig, "plots/training_history.png")
```

```python Save Evaluation Results
import mlflow
import pandas as pd
from sklearn.metrics import classification_report

with mlflow.start_run():
    predictions = model.predict(X_test)
    
    # Create classification report
    report = classification_report(
        y_test, predictions, output_dict=True
    )
    
    # Log as JSON
    mlflow.log_dict(report, "results/classification_report.json")
    
    # Log as table
    report_df = pd.DataFrame(report).transpose()
    mlflow.log_table(report_df, "results/classification_table.json")
```
