    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with MLflow in minutes
  </p>

  <Steps>
    <Step title="Install MLflow">
      Install the MLflow Python package using pip:
      
      ```bash
      pip install mlflow
      ```
    </Step>
    
    <Step title="Start the MLflow UI">
      Launch the MLflow tracking server and UI:
      
      ```bash
      mlflow server
      ```
      
      Navigate to `http://localhost:5000` to view the MLflow UI.
    </Step>
    
    <Step title="Track your first experiment">
      Create a simple tracking script:
      
      ```python
      import mlflow
      from sklearn.ensemble import RandomForestRegressor
      from sklearn.datasets import load_diabetes
      from sklearn.model_selection import train_test_split

      mlflow.sklearn.autolog()

      db = load_diabetes()
      X_train, X_test, y_train, y_test = train_test_split(db.data, db.target)

      rf = RandomForestRegressor(n_estimators=100, max_depth=6)
      rf.fit(X_train, y_train)
      ```

      MLflow automatically logs parameters, metrics, and the trained model.
    </Step>

  </Steps>
