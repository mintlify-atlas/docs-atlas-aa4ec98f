## Quick Installation

Install MLflow using pip:

```bash
pip install mlflow
```

<Note>
MLflow requires Python 3.10 or later. Verify your Python version with `python --version`
</Note>

## Your First MLflow Experiment

Let's track a simple machine learning experiment using scikit-learn with MLflow's autologging feature.

<Steps>
  <Step title="Create a training script">
    Create a file called `train.py` with the following code:

    ```python train.py
    import mlflow
    from sklearn.datasets import load_iris
    from sklearn.linear_model import LogisticRegression
    from sklearn.model_selection import train_test_split

    # Enable MLflow's automatic experiment tracking for scikit-learn
    mlflow.sklearn.autolog()

    # Load the Iris dataset
    X, y = load_iris(return_X_y=True)
    X_train, X_test, y_train, y_test = train_test_split(X, y, random_state=42)

    # Train a Logistic Regression model
    model = LogisticRegression(max_iter=200)
    
    with mlflow.start_run():
        model.fit(X_train, y_train)
        
    print("Training complete! View results in the MLflow UI.")
    ```

    <Note>
    The `mlflow.sklearn.autolog()` call automatically logs parameters, metrics, and the model without any additional code!
    </Note>
  </Step>

  <Step title="Run the training script">
    Execute your script:

    ```bash
    python train.py
    ```

    You should see output confirming the training completed successfully.
  </Step>

  <Step title="Start the MLflow UI">
    Launch the MLflow tracking UI:

    ```bash
    mlflow ui
    ```

    Or use the server command for more options:

    ```bash
    mlflow server
    ```

    <Note>
    By default, the UI runs at http://localhost:5000
    </Note>
  </Step>

  <Step title="View your results">
    Open your browser and navigate to http://localhost:5000

    In the MLflow UI, you'll see:
    - **Parameters**: Model hyperparameters (e.g., max_iter=200)
    - **Metrics**: Training accuracy, precision, recall, f1-score
    - **Artifacts**: The trained model, confusion matrix, and more
    - **Tags**: Metadata about the run
  </Step>
</Steps>

## Tracking LLM Applications

MLflow also provides powerful tracing capabilities for LLM applications. Here's a quick example with OpenAI:

```python llm_trace.py
import os
import mlflow
from openai import OpenAI

# Set your OpenAI API key
os.environ["OPENAI_API_KEY"] = "your-api-key-here"

# Enable tracing for OpenAI
mlflow.openai.autolog()

client = OpenAI()

# Make a request - it will be automatically traced
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Tell me a joke about Python"}],
    temperature=0.7,
)

print(response.choices[0].message.content)
```

<Warning>
Don't forget to replace `"your-api-key-here"` with your actual OpenAI API key, or set it as an environment variable.
</Warning>

Navigate to the "Traces" tab in the MLflow UI to see:
- Request/response details
- Token usage and costs
- Latency metrics
- Full conversation history

## Manual Logging Example

For more control, you can manually log parameters, metrics, and artifacts:

```python manual_logging.py
import mlflow
from random import random, randint

# Start an MLflow run
with mlflow.start_run():
    # Log parameters
    mlflow.log_param("learning_rate", 0.01)
    mlflow.log_param("batch_size", 32)
    
    # Log metrics over multiple steps
    for epoch in range(10):
        mlflow.log_metric("loss", random(), step=epoch)
        mlflow.log_metric("accuracy", random(), step=epoch)
    
    # Log artifacts (files)
    with open("model_summary.txt", "w") as f:
        f.write("Model training summary")
    mlflow.log_artifact("model_summary.txt")
    
    print("Run completed!")
```

## Working with Logged Models

Retrieve and use models logged by autologging:

```python load_model.py
import mlflow
import pandas as pd
from sklearn.datasets import load_iris

# Get the most recently logged model
model = mlflow.last_logged_model()
print(f"Model URI: {model.model_uri}")
print(f"Model params: {model.params}")
print(f"Model metrics: {model.metrics}")

# Load the model for inference
loaded_model = mlflow.sklearn.load_model(model.model_uri)

# Make predictions
X, y = load_iris(return_X_y=True)
sample_data = X[:5]
predictions = loaded_model.predict(sample_data)
print(f"Predictions: {predictions}")
```

## Common Commands Reference

<CodeGroup>
```bash Start UI
mlflow ui
```

```bash Start Server
mlflow server --host 0.0.0.0 --port 5000
```

```bash Run Project
mlflow run . -P param1=value1
```

```bash Search Runs
mlflow runs list --experiment-id 0
```
</CodeGroup>

<Note>
All MLflow data is stored locally in the `mlruns` directory by default. You can configure a remote tracking server for team collaboration.
</Note>
