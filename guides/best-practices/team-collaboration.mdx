## Overview

Successful machine learning projects require effective collaboration between data scientists, ML engineers, and stakeholders. MLflow provides features that enable teams to share experiments, models, and insights efficiently.

## Workspace Organization

### Structure Workspaces by Team or Project

Organize workspaces to match your team structure:

```python
import mlflow

# Create team-specific workspaces
mlflow.create_workspace(
    name="data-science-team",
    description="Core data science team experiments"
)

mlflow.create_workspace(
    name="ml-platform-team",
    description="ML platform and infrastructure experiments"
)

mlflow.create_workspace(
    name="product-recommendations",
    description="Product recommendation project workspace"
)
```

<Accordion title="Workspace organization strategies">
Common workspace patterns:
- **By team**: Separate workspaces for each team (NLP, Computer Vision, etc.)
- **By project**: One workspace per major project or product
- **By environment**: Separate dev, staging, production workspaces
- **Hybrid**: Team workspaces with project sub-organization via experiment naming
</Accordion>

### Set Workspace Context

Make it easy for team members to work in the right workspace:

```python
import mlflow

# Set active workspace
mlflow.set_workspace("product-recommendations")

# List available workspaces

for ws in workspaces:
    print(f"{ws.name}: {ws.description}")

# Get current workspace
current_workspace = mlflow.get_workspace()
```


<Accordion title="Essential experiment tags">
Recommended tags for team collaboration:

## Model Registry Collaboration

### Establish Model Promotion Workflows

Define clear stages for model progression:

```python
from mlflow.tracking import MlflowClient

client = MlflowClient()

class ModelPromotion:
    """Standardized model promotion workflow"""
    
    def request_staging(self, model_name, version, requester):
        """Request promotion to staging"""
        client.set_model_version_tag(
            name=model_name,
            version=version,
            key="promotion_request",
            value="staging"
        )
        client.set_model_version_tag(
            name=model_name,
            version=version,
            key="requested_by",
            value=requester
        )
        print(f"Staging promotion requested for {model_name} v{version}")
    
    def approve_staging(self, model_name, version, approver, metrics_validated=True):
        """Approve and promote to staging"""
        if not metrics_validated:
            raise ValueError("Metrics must be validated before promotion")
        
        # Set staging alias
        client.set_registered_model_alias(
            name=model_name,
            alias="staging",
            version=version
        )
        
        client.set_model_version_tag(
            name=model_name,
            version=version,
            key="staging_approved_by",
            value=approver
        )
        client.set_model_version_tag(
            name=model_name,
            version=version,
            key="staging_date",
            value=pd.Timestamp.now().isoformat()
        )
        print(f"{model_name} v{version} promoted to staging")

promotion = ModelPromotion()
promotion.request_staging("fraud_detector", "5", "alice@company.com")
promotion.approve_staging("fraud_detector", "5", "bob@company.com")
```

### Implement Model Review Checklists

Create standardized review processes:

```python
import mlflow
from mlflow.tracking import MlflowClient

def review_model_checklist(model_name, version):
    """Model review checklist"""
    client = MlflowClient()
    
    checklist = {
        "metrics_validated": False,
        "bias_tested": False,
        "fairness_evaluated": False,
        "edge_cases_tested": False,
        "documentation_complete": False,
        "dependencies_reviewed": False,
        "security_scanned": False,
        "performance_benchmarked": False
    }
    
    # Log checklist
    model_version = client.get_model_version(model_name, version)
    with mlflow.start_run(run_id=model_version.run_id):
        mlflow.log_dict(checklist, "review_checklist.json")
    
    return checklist

# After review
def complete_review(model_name, version, reviewer, checklist):
    """Complete model review"""
    client = MlflowClient()
    
    if not all(checklist.values()):
        incomplete = [k for k, v in checklist.items() if not v]
        raise ValueError(f"Review incomplete. Missing: {incomplete}")
    
    client.set_model_version_tag(
        name=model_name,
        version=version,
        key="review_complete",
        value="true"
    )
    client.set_model_version_tag(
        name=model_name,
        version=version,
        key="reviewed_by",
        value=reviewer
    )
```

<Accordion title="Model review checklist items">
Comprehensive review should include:
- **Metrics validation**: Performance meets requirements
- **Bias testing**: Fairness across demographic groups
- **Edge cases**: Handles unusual inputs gracefully
- **Documentation**: Model card, API docs, deployment guide
- **Dependencies**: All dependencies documented and approved
- **Security**: No known vulnerabilities in dependencies
- **Performance**: Latency and throughput acceptable
- **Monitoring**: Alerts and dashboards configured
</Accordion>

### Create Reusable Training Templates

Share common training patterns:

```python
import mlflow
from abc import ABC, abstractmethod

class MLflowExperiment(ABC):
    """Base class for team experiments"""
    
    def __init__(self, experiment_name, owner):
        self.experiment_name = experiment_name
        self.owner = owner
        mlflow.set_experiment(experiment_name)
    
    def run(self, run_name, params):
        """Run experiment with standard setup"""
        with mlflow.start_run(run_name=run_name):
            # Standard setup
            mlflow.set_tags({
                "owner": self.owner,
                "framework": self.get_framework(),
                "experiment_type": self.__class__.__name__
            })
            mlflow.log_params(params)
            
            # Custom training logic
            metrics = self.train(params)
            mlflow.log_metrics(metrics)
            
            # Standard logging
            model = self.get_model()
            mlflow.sklearn.log_model(model, "model")
            
            return metrics
    
    @abstractmethod
    def train(self, params):
        """Implement training logic"""
        pass
    
    @abstractmethod
    def get_model(self):
        """Return trained model"""
        pass
    
    @abstractmethod
    def get_framework(self):
        """Return framework name"""
        pass

# Usage
class ChurnPredictionExperiment(MLflowExperiment):
    def train(self, params):
        # Training logic
        return {"accuracy": 0.92}
    
    def get_model(self):
        return trained_model
    
    def get_framework(self):
```

### Share Common Utilities

Create shared libraries for common operations:

```python
# team_mlflow_utils.py
import mlflow
from mlflow.tracking import MlflowClient

class TeamMLflowUtils:
    """Shared utilities for team MLflow operations"""
    
    @staticmethod
    def find_best_run(experiment_name, metric="accuracy", ascending=False):
        """Find best run in experiment by metric"""
        experiment = mlflow.get_experiment_by_name(experiment_name)
        runs = mlflow.search_runs(
            experiment_ids=[experiment.experiment_id],
            order_by=[f"metrics.{metric} {'ASC' if ascending else 'DESC'}"]
        )
        return runs.iloc[0] if len(runs) > 0 else None
    
    @staticmethod
    def compare_runs(run_ids, metrics):
        """Compare multiple runs across metrics"""
        client = MlflowClient()
        comparison = []
        
        for run_id in run_ids:
            run = client.get_run(run_id)
            run_data = {"run_id": run_id, "run_name": run.data.tags.get("mlflow.runName")}
            for metric in metrics:
                run_data[metric] = run.data.metrics.get(metric)
            comparison.append(run_data)
        
        return pd.DataFrame(comparison)
    
    @staticmethod
    def clone_run(run_id, new_params=None):
        """Clone a run with optional parameter changes"""
        client = MlflowClient()
        source_run = client.get_run(run_id)
        
        with mlflow.start_run():
            # Copy params
            params = dict(source_run.data.params)
            if new_params:
                params.update(new_params)
            mlflow.log_params(params)
            
            # Copy tags
            for key, value in source_run.data.tags.items():
                if not key.startswith("mlflow."):
                    mlflow.set_tag(key, value)
            mlflow.set_tag("cloned_from", run_id)
```

```python
import mlflow

def generate_experiment_url(experiment_name, filter_string=None):
    """Generate shareable experiment URL with filters"""
    experiment = mlflow.get_experiment_by_name(experiment_name)
    base_url = mlflow.get_tracking_uri()
    url = f"{base_url}/#/experiments/{experiment.experiment_id}"
    
    if filter_string:
        # URL encode filter
        import urllib.parse
        encoded_filter = urllib.parse.quote(filter_string)
        url += f"?searchFilter={encoded_filter}"
    
    return url
```

## Code Review and Reproducibility

### Log Git Information

Automatically track code versions:

```python
import mlflow
import subprocess

def get_git_info():
    """Get current git information"""
    try:
        commit = subprocess.check_output(['git', 'rev-parse', 'HEAD']).decode('utf-8').strip()
        branch = subprocess.check_output(['git', 'rev-parse', '--abbrev-ref', 'HEAD']).decode('utf-8').strip()
        remote = subprocess.check_output(['git', 'config', '--get', 'remote.origin.url']).decode('utf-8').strip()
        
        # Check for uncommitted changes
        status = subprocess.check_output(['git', 'status', '--porcelain']).decode('utf-8')
        is_dirty = len(status) > 0
        
        return {
            "git_commit": commit,
            "git_branch": branch,
            "git_remote": remote,
            "git_dirty": str(is_dirty)
        }
    except:
        return {}

with mlflow.start_run():
    # Automatically log git info
    git_info = get_git_info()
    mlflow.set_tags(git_info)
    
    # Your training code
    model = train_model()
```
